Movie Vault

Problem?

I have an issue with remembering what movies I want to watch, movies I have already watched, and what I thought about them.

Solution?

Create a desktop app which I can add movies, give them tags including, watched/unwatched, movie category, rating, and any other tag I might want to add.

Plan

I plan to use React and Electron for my UI, Node.js for backend, and SQLite for my database. I plan to add features such as syncing with a phone app, and streaming to other devices.
